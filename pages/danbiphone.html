🚀 빠른 처리 순서
  1. 자마드 티켓 제목 작성예: [단비폰] CS 명칭 / 교사명 / 증상 대략 작성
  2. 선생님 부재 시 대응담당 팀장에게 선생님과 라인방 만들어 달라고 요청 또는 업무폰 번호로 직접 연락 시도
  3. 필수 어플 설치 안내RemoteCall + RSAssistant 설치 안내. 파일 확인이 필요하면 파일관리자+도 안내
  4. 원격 연결 안내RSAssistant 권한 허용 후 모바일 지원에서 6자리 숫자 입력 안내
  5. 점검 및 기록증상에 따라 시작/종료시간 기록 → 조치 내용 자마드 티켓 댓글로 기록
  6. 2차 점검 필요 시아래 "2차 점검 요청서 만들기"에서 체크 후 개발 전달 양식 복사
