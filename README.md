# 단비공감팀 상담가이드 v12

## 이번 버전 핵심

- 좌측 사이드바 콤팩트 모드 적용
  - 기본값: 아이콘만 보이는 72px 폭
  - 펼침 시: 284px 폭으로 상세 메뉴 표시
  - 좌측 상단 화살표 버튼으로 접기/펼치기 가능

- 좌측 메뉴 구조 개편
  - 홈
  - 점검 운영 현황
  - 기술상담 아코디언
  - 일반상담 아코디언
  - `펼치기 · 접기` 컨트롤 추가
  - 기본 상태에서는 기술상담/일반상담 모두 닫힘

- 중앙 가이드 화면 전면 개편
  - 테이블형 제거
  - 둥근 카드형 가이드 리스트 적용
  - 카드 구성: 아이콘 / 문제 제목 / 3줄 요약 조치법 / 태그 / 즐겨찾기
  - 적용기기, 처리방향/처리유형, 키워드 검색 필터 적용

- 우측 영역 정리
  - 최근 업데이트 제거
  - 자료 다운로드 제거
  - 더 빠르게 찾는 팁 제거
  - 즐겨찾기, 자주 쓰는 문자 중심으로 정리

- 테마
  - 우측 상단 라이트/다크 토글 스위치 적용
  - localStorage로 마지막 선택 테마 유지
