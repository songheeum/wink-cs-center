# 단비공감팀 기술상담 가이드

## 소개

단비공감팀 상담사들이 기술상담 문의를 빠르게 확인할 수 있도록 제작한 상담 가이드입니다.

자주 문의되는 기술상담 내용을 검색하거나 카테고리별로 확인할 수 있으며, 단계별 안내와 참고 이미지를 함께 제공합니다.

---

## 서비스 주소

https://songheeum.github.io/wink-cs-center/

---

## 제공 내용

현재 아래 항목에 대한 기술상담 가이드를 제공합니다.

* 윙크봇
* 윙크스쿨
* 삼성 태블릿
* 레노버 K11
* 레노버 P11

---

## 가이드 종류

### 참고형 가이드

기기 기능이나 버튼 설명 등 참고용 자료를 제공합니다.

예시

* 학습기 각 버튼 및 기능
* 상태 표시등 안내
* 기기 구성 설명

---

### 절차형 가이드

고객에게 안내해야 하는 순서를 단계별로 제공합니다.

예시

* TV 연결
* 공장 초기화
* 원격 연결
* 와이파이 설정

---

## 가이드 수정 방법

가이드 내용은 Google Sheet에서 관리됩니다.

내용 수정 후 저장하면 사이트에 자동 반영됩니다.

### 수정 순서

1. Google Sheet 접속
2. 내용 수정 또는 신규 등록
3. 저장
4. 사이트에서 확인

---

## 이미지 등록 방법

가이드에 사용할 이미지는 아래 폴더에 업로드합니다.

assets/guides/

예시

winkbot/winkbot_button_01.png

시트에는 파일 경로만 입력하면 자동으로 연결됩니다.

---

## 데이터 관리

### 운영 데이터

Google Sheet (CSV)

사이트에서 사용하는 실제 데이터입니다.

### 기술상담 백업 데이터

data/technical-faq-backup.json

Google Sheet 연결에 문제가 발생했을 경우 사용할 비상 백업 파일입니다.

### 일반상담 데이터

data/general-faq.json

향후 일반상담 FAQ 기능 확장을 위해 준비된 파일입니다.

---

## 사이트가 정상적으로 보이지 않을 경우

1. 새로고침(F5)
2. 잠시 후 재접속
3. 관리자에게 문의

Google Sheet 연결 오류 발생 시 백업 데이터를 사용하도록 설정되어 있습니다.
