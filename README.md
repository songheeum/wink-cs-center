# 단비공감팀 상담가이드 · 점검 대시보드

GitHub Pages용 정적 사이트. 상담가이드 문서 뷰어 + 자마드 점검 운영 대시보드.

## 파일
- `index.html` · `style.css` · `app.js` · `app.json`

## 데이터 연동 (app.json)
- `csvUrl` : 점검 운영 대시보드용 CSV (접수일/상태/처리자 등)
- `guides.sources` : 상담가이드 CSV 2개 (기술상담 gid=475812845, 일반상담 gid=243492375)
- `guides.imageBase` : 이미지 폴더 경로(기본 `img/`). 시트 `이미지` 칸에 파일명만 넣으면 자동 경로 조합
- `refreshSeconds` : 대시보드 + 상담가이드 자동 갱신 주기(현재 30초)

## 가이드 시트 컬럼 (실제 헤더 기준 자동 매핑)
| 시트 헤더 | 화면 위치 |
|---|---|
| 가이드명 | 문서 제목 |
| 증상그룹 / 업무그룹 | 좌측 트리 분류(중분류) · breadcrumb |
| 세부분류 | breadcrumb 말단 · 태그 |
| 적용기기 | 태그 |
| 고객표현 | 「고객 표현」 |
| 확인사항 | 「상담사가 먼저 확인할 것」 |
| 조치방법 | 「조치 방법」(번호 목록) |
| 처리기준 | 「처리 기준」(일반상담) |
| 안내멘트 | 「안내 멘트」(복사 버튼) |
| 주의사항 | 「주의사항」(경고 박스) |
| 태그 | 문서 상단 #태그 |
| 이미지 | 「참고 이미지」 |

- 여러 항목은 셀 안에서 **`|`(파이프)** 로 구분 → 목록으로 자동 분리
- 내용 안의 `·` `/` 는 구분자로 쓰지 않음(그대로 표시)
- 값이 빈 컬럼의 섹션은 자동으로 숨김

## 유지된 핵심 기능
Google Sheet CSV 실시간 연동 · FAQ 검색 · 카테고리 트리 · 상세 문서 · 이미지 출력 · 기술/일반상담 · 대시보드 집계 · 개인정보 컬럼 제외

## 참고
- Pretendard는 jsdelivr CDN 사용(사내망 차단 시 시스템 폰트로 폴백)
- 대시보드 CSV가 접근 불가하면 자동으로 GViz JSONP 폴백 시도

## 이번 수정 반영
- 기술상담/일반상담 CSV를 `guides.sources`로 연결했고, 30초마다 상담가이드 데이터까지 다시 불러옵니다.
- 검색어 입력 시 큰 히어로 배너는 숨기고 바로 검색 결과 카드만 표시합니다.
- 검색 결과 카드/카테고리 카드 클릭 시 상세 가이드로 이동하도록 연결했습니다.
- 좌측 기술상담/일반상담 제목 줄 전체를 눌러 펼치고 접을 수 있게 수정했습니다.
- 검색어가 있어도 좌측 메뉴는 필터링하지 않고 전체 메뉴를 유지합니다.


## 최종 운영본 메모

- 기술상담/일반상담 가이드는 Google Sheets 게시 CSV와 30초 주기로 연동됩니다.
- 점검 운영 현황 필터는 화면 진입/이탈 시 초기화되며, 화면 내에서는 필터 초기화 버튼으로 전체 기준으로 복귀할 수 있습니다.
- 좌측 메뉴는 기본 메뉴와 상담 가이드 영역을 분리했고, 기술상담/일반상담 색상 위계를 구분했습니다.
