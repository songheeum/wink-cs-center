# wink-cs-center
# 단비공감팀 기술상담 가이드

## 이 사이트는 무엇인가요?

단비공감팀 상담사들이 기술상담 문의를 빠르게 확인할 수 있도록 만든 기술상담 가이드입니다.

자주 문의되는 내용들을 검색하거나 카테고리별로 확인할 수 있으며, 상담 시 필요한 안내 절차와 참고 이미지를 쉽게 확인할 수 있습니다.

---

## 서비스 주소

https://songheeum.github.io/wink-cs-center/

---

## 어떤 내용을 볼 수 있나요?

현재 아래 기기 및 서비스 관련 가이드를 제공하고 있습니다.

* 윙크봇
* 윙크스쿨
* 삼성 태블릿
* 레노버 K11
* 레노버 P11

---

## 가이드 종류

### 참고형 가이드

기기 버튼 설명, 기능 안내 등 참고용 자료입니다.

예시

* 학습기 각 버튼 및 기능
* 상태 표시등 안내
* 기기 구성 설명

---

### 절차형 가이드

고객에게 안내해야 하는 순서를 단계별로 제공합니다.

예시

* TV 연결
* 공장 초기화
* 원격 연결
* 와이파이 설정

---

## 내용 수정은 어떻게 하나요?

기술상담 가이드는 Google Sheet에서 관리됩니다.

별도의 GitHub 작업 없이 시트 내용만 수정하면 사이트에 자동 반영됩니다.

### 수정 순서

1. Google Sheet 열기
2. 내용 수정 또는 신규 등록
3. 저장
4. 사이트에서 확인

---

## 이미지 등록 방법

가이드에 사용할 이미지는 지정된 폴더에 업로드합니다.

이미지 업로드 후 시트에 파일명을 입력하면 자동으로 연결됩니다.

---

## 사이트가 정상적으로 보이지 않는 경우

1. 페이지 새로고침(F5)
2. 잠시 후 다시 접속
3. 관리자에게 문의

시트 연결에 문제가 발생할 경우 백업 데이터를 사용하도록 설정되어 있습니다.
